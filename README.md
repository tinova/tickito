## Install

### Ticker Source

You'll need an API key from [CyptoCompare](https://www.cryptocompare.com "CyptoCompare").

### Setting Up

```sh
$ pip3 install adafruit-circuitpython-ssd1306 pyyaml
$ git clone https://github.com/tinova/tickito.git
 ```
